# Octra

Octra testnet rehberimiz yayında! 

Octra, 2021’de İsviçre merkezli olarak kurulan ve bugün Fully Homomorphic Encryption teknolojisini kullanarak gizliliği en üst seviyeye taşıyan yeni nesil bir blockchain ağı. 

Octra, DeFi dünyasında zincirler arası takasları daha hızlı, güvenli ve kullanıcı dostu hale getirmeyi amaçlıyor. Bu erken aşamada testnet sürecinde yer alarak ekosisteme ilk katılanlardan biri olabilirsin!

Katılım için sadece bir GitHub hesabı yeterli.
